# QL-BD-CHECKIN - 百度网盘自动化 (青龙面板)

适用于 [青龙面板](https://github.com/whyour/qinglong) 的百度网盘自动化脚本。

## 功能特性 (v4.0)

模拟百度网盘APP日常任务，每日自动执行6项任务：

| 任务 | 说明 | 状态 |
|------|------|------|
| 签到尝试 | 每日尝试调用签到API | ⚠️ API已废弃 |
| 会员信息 | 查询等级、成长值、会员状态 | ✅ 可用 |
| 成长值历史 | 查看近期成长值变化记录 | ✅ 可用 |
| 金币系统 | 查询金币余额和记录 | ✅ 可用 |
| 网盘容量 | 查询总容量和已使用空间 | ✅ 可用 |
| 会话保活 | 保持BDUSS会话活跃 | ✅ 可用 |

其他特性：
- **STOKEN自动获取** - 通过BDUSS调用PCS API自动获取，无需手动配置
- **BDUSS有效性验证** - 自动检测Cookie是否过期
- **多账号支持** - 批量处理多个账号
- **通知推送** - 青龙面板通知

## 快速开始

### 1. 添加依赖

```bash
pip install requests rsa
```

### 2. 配置环境变量

#### 方式一：BDUSS Cookie（推荐）

| 变量名 | 值 | 说明 |
|--------|-----|------|
| `BDUSS` | 你的BDUSS值 | 多账号用 `&` 分隔 |
| `STOKEN` | 你的STOKEN值（可选） | 不填则自动获取 |

#### 方式二：账号密码

| 变量名 | 值 | 说明 |
|--------|-----|------|
| `BAIDU_ACCOUNTS` | `手机号#密码` | 多账号用 `&` 分隔 |

### 3. 添加定时任务

| 字段 | 值 |
|------|------|
| 名称 | 百度网盘签到 |
| 命令 | `task baidu_pan_checkin.py` |
| 定时规则 | `0 8 * * *` |

### 4. 获取BDUSS

#### 方法一：浏览器开发者工具（推荐）

1. 浏览器访问 [pan.baidu.com](https://pan.baidu.com) 并登录
2. 按 `F12` → `Application` → `Cookies` → `BDUSS`
3. 复制BDUSS的值

#### 方法二：Chrome浏览器插件

本仓库包含Chrome/Edge浏览器插件，可一键获取BDUSS：
1. 下载 `bduss-extension/` 文件夹
2. Chrome打开 `chrome://extensions/` → 开启开发者模式
3. 加载已解压的扩展程序 → 选择 `bduss-extension/`

## 常见问题

### 签到显示"API返回空响应"？

百度已废弃Web端签到API，签到功能已迁移至百度网盘APP。本脚本仍会每日尝试签到并监控账户状态。

### BDUSS失效怎么办？

BDUSS有效期较长，但可能会因修改密码、异地登录等原因失效。失效后请重新获取。

### STOKEN需要手动配置吗？

不需要。脚本会通过BDUSS自动调用PCS API获取STOKEN。
