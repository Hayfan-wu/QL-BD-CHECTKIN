# Auto.js 百度网盘APP任务自动化指南

## 概述

本方案使用 Auto.js Pro 在你的 Android 手机上自动完成百度网盘APP任务中心的日常任务。

### 与青龙面板的分工

```
服务器端（青龙面板）              手机端（Auto.js）
┌──────────────────┐           ┌──────────────────┐
│ baidu_pan_       │           │ baidu_pan_       │
│ checkin.py       │           │ autojs.js        │
│                  │           │                  │
│ ✅ BDUSS验证     │           │ ✅ 观看广告×15   │
│ ✅ 会员信息监控  │           │ ✅ 添加小组件    │
│ ✅ 签到尝试      │           │ ✅ 美团刷视频    │
│ ✅ 成长值历史    │           │ ✅ 每日定时执行  │
│ ✅ 网盘容量      │           │                  │
│ ✅ 会话保活      │           │                  │
└──────────────────┘           └──────────────────┘
        ↑                              ↑
    每日8:00执行                   每日8:30执行
```

## 第一步：安装 Auto.js

### 方式一：安装 Auto.js Pro（推荐）

1. 在手机浏览器打开 https://pro.autojs.org/
2. 下载并安装 Auto.js Pro APK
3. 打开 Auto.js Pro
4. 点击右上角菜单 → **设置**
5. 开启 **无障碍服务** → 选择 Auto.js Pro → 开启

### 方式二：安装 Auto.js 6（免费开源）

1. 下载地址：https://github.com/SuperMonster003/AutoJs6/releases
2. 下载最新的 APK 文件
3. 安装后开启无障碍服务

> **注意**：Auto.js 6 是免费开源版本，功能与 Pro 版基本一致。如果 Pro 版无法获取，使用 Auto.js 6 即可。

## 第二步：导入脚本

1. 将 `baidu_pan_autojs.js` 文件传输到手机
   - 可通过微信/QQ发送到手机
   - 或通过数据线拷贝
   - 或在手机浏览器直接访问 GitHub 仓库下载
2. 打开 Auto.js
3. 点击右上角 **+** 号 → **导入**
4. 选择 `baidu_pan_autojs.js` 文件
5. 脚本已导入

## 第三步：测试运行

**首次运行前，请手动操作一次：**

1. 手动打开百度网盘APP
2. 进入「我的」→ 找到「做任务赚积分」
3. 观察任务中心的界面布局
4. 记录各按钮的文字（如果与脚本不同，需要修改）

**测试脚本：**

1. 在 Auto.js 中打开 `baidu_pan_autojs.js`
2. 点击右上角 **▶ 运行** 按钮
3. 首次运行会请求 **截图权限**，点击允许
4. 脚本将自动执行：
   - 启动百度网盘
   - 进入任务中心
   - 观看广告1次（测试模式）
   - 尝试添加小组件
   - 尝试美团任务

5. 观察执行日志，如果有报错请记录

**调试模式：**

如果脚本无法正确识别按钮，可以在脚本开头修改 `CONFIG.DEBUG = true`，然后查看详细日志。

## 第四步：配置定时任务

### 方式一：使用 Auto.js Pro 内置定时（推荐）

在 Auto.js Pro 中运行以下命令（在脚本编辑器中执行）：

```javascript
// 在脚本末尾添加这行，然后运行一次
engines.execScriptFile("baidu_pan_autojs.js", {
    arguments: { mode: "setup" }
});
```

或直接在 Auto.js Pro 的界面中设置：
1. 长按脚本 → **定时任务**
2. 设置执行时间：每天 08:30
3. 保存

### 方式二：使用 Android 系统闹钟（通用）

1. 设置手机闹钟：每天 08:25
2. 闹钟响起后，手动打开 Auto.js 运行脚本
3. 脚本执行完毕后手机会自动锁屏

### 方式三：使用 Tasker（进阶）

如果你安装了 Tasker，可以配置：
1. Tasker 定时触发（每天 08:30）
2. 执行 Auto.js 脚本（通过 Auto.js 的插件）
3. 完成后自动锁屏

## 第五步：保持手机常开

为确保定时任务能执行，需要：

1. **手机不要关机**，保持充电
2. **Auto.js 加入电池优化白名单**：
   - 设置 → 电池 → 不受限制 → 添加 Auto.js
3. **Auto.js 保持后台运行**：
   - 在 Auto.js 设置中开启「保持后台运行」
   - 在最近任务列表中锁定 Auto.js
4. **手机保持联网**

## 界面适配说明

百度网盘APP的界面可能会更新，如果脚本无法正确识别按钮，需要调整选择器。

### 查看当前界面元素

在 Auto.js 中使用 **布局分析** 功能：
1. 打开 Auto.js → 点击 **布局分析**
2. 然后打开百度网盘任务中心页面
3. 查看各按钮的 text、desc、id 属性
4. 根据实际情况修改脚本中的选择器

### 常见适配修改

**1. "我的"标签找不到：**

打开脚本，找到 `navigateToTaskCenter` 函数，修改：
```javascript
// 如果"我的"的文字不是"我的"，比如是"个人中心"
var myTab = text("个人中心").findOne(5000);
```

**2. 任务中心入口找不到：**

```javascript
// 如果入口文字不同
var taskEntryTexts = [
    "你的实际入口文字",  // 替换为实际看到的文字
    "做任务赚积分",
    "任务中心",
];
```

**3. 广告关闭按钮找不到：**

如果广告无法自动关闭，脚本会尝试多种方式。如果都不行，可以增加坐标点击：
```javascript
// 修改 closeAd 函数中的坐标
// 右上角关闭按钮的坐标（根据你的手机分辨率调整）
var width = device.width;
click(width - 50, 80);  // 调整这两个数字
```

## 任务执行顺序

```
脚本启动
    │
    ├─ 1. 启动百度网盘APP
    │
    ├─ 2. 进入「我的」→「做任务赚积分」
    │
    ├─ 3. 观看广告×15次（约10-15分钟）
    │     ├─ 点击「观看广告」
    │     ├─ 等待35秒（广告播放）
    │     ├─ 点击「关闭」
    │     └─ 重复15次
    │
    ├─ 4. 添加桌面小组件
    │     ├─ 点击「添加小组件」
    │     └─ 点击「添加/确定」
    │
    ├─ 5. 美团刷视频
    │     ├─ 点击「去美团刷视频」
    │     ├─ 等待跳转美团
    │     ├─ 上滑切换视频×5次
    │     └─ 返回百度网盘
    │
    └─ 6. 发送通知 + 返回桌面
```

## 常见问题

### Q: 脚本提示"无障碍服务未开启"

A: 设置 → 无障碍 → 找到 Auto.js → 开启。部分手机需要额外开启"使用情况访问权限"。

### Q: 广告看完后没有自动关闭

A: 广告关闭按钮的位置可能不同。请用 Auto.js 的布局分析功能查看广告界面的关闭按钮，然后修改 `closeAd` 函数中的选择器。

### Q: "我的"标签点击不了

A: 可能是底部标签栏的文字不是"我的"。用布局分析查看实际文字，修改脚本中的 `text("我的")` 为实际文字。

### Q: 美团任务失败

A: 需要手机上安装美团APP并登录。如果未安装美团，此任务会自动跳过。

### Q: 定时任务不执行

A: 
1. 检查手机是否关机
2. 检查 Auto.js 是否在后台运行
3. 检查电池优化设置
4. 尝试使用方式三（Tasker）

### Q: 脚本运行中途停止

A: 可能是手机自动锁屏或内存不足。请：
1. 在 Auto.js 设置中开启「保持屏幕常亮」
2. 在开发者选项中设置「不锁定屏幕」（充电时）
3. 锁定 Auto.js 在最近任务中

## 注意事项

- 脚本仅供学习交流，请遵守百度网盘用户协议
- 短时间内频繁操作可能触发风控，脚本已内置随机延迟
- APP更新后可能需要调整选择器
- 观看广告任务依赖网络状况，网络差时广告可能加载失败
- 建议每周手动检查一次任务完成情况
